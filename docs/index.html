About This Project

This is a system to track the inventory of donated and purchased items for the HRDC in southwest Montana. A demo video showing usage of the application can be found here.

Features

  • Administrators can add new items and can update existing items
  • Any signed in user can update the current inventory level
  • Alerts for low inventory levels in dashboard and via email
  • Users can export the inventory data to a CSV file
  • Charts and reports generated from the inventory.
